Game: Build and Conquer
Team: TEKC
==================================================================
Description: Build and Conquer is a mobile platform game in which 2 players are up against each other. Each player can place building, each building can produce a unit that automatically seeks its enemies and attack them.
==================================================================
Controls:
Left Mouse Click - to select building and place it on your side.

Game's detail:

Gold - currency to build your building; gold is increased overtime.
